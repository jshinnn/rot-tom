# Project Title

Data explainer project for CS 448B.

## Description

`clean_data` holds critic reviews with the following attributes: 
- rotten tomatoes link
- critic name
- top critic (T/F)
- publisher name
- review type
- review score
- review date
- review content
- gender

`raw_data` holds source data.
